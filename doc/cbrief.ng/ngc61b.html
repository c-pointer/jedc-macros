inq_top_left ([top_line], [top_col], [win_id], [line], [col], [buf_id])

     Usage: Get buffer positioning information.

Parameters: Top_line and top_col .ints. are filled with position of the
            top left character in the buffer in the window given by win_id
            .int.. If win_id is omitted, the current window is used.
            Line and col .ints. give the current buffer (for that window).
            Buf_id .int. is set to the buffer id attached to the window.

   Returns: Nothing
See Also: set_top_left
