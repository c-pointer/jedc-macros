pause_on_error ([pause])

     Usage: Set pausing execution on every error.  This is mainly a
            debug feature, used in conjunction with the error macro
            and to catch error messages before the vanish from the
            screen.

Parameters: Pause .int. is non-0 to enable pausing and 0 to
            disable. If omitted, the current pause state is toggled.

   Returns: The previous state.
See Also: error
