change_window ({direction})

     Usage: Makes a different window the active window.  Note that this
            works only on tiled windows.

Parameters: Direction .int. indicates the direction of the new
            window: 0=above, 1=right, 2=below, 3=left. If omitted,
            the user is prompted for an arrow key.

   Returns: Greater than 0 for success, 0 or less for failure.
See Also: set_window next_window
