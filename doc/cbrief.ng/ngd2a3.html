int_to_key (key_integer)

     Usage: Convert an integer to mnemonic key string.

Parameters: Key_integer .int. is a keystroke value like that returned
            by read_char.

   Returns: The mnemonic key string (like "<Ctrl-l>").  This type of
            string is used by assign_to_key. See Key Table for a
            full description of the format of key strings.
