autoload (fname, macro1, [macro2 ...])

     Usage: Notify BRIEF where to locate macros and globals if they
            are not already in memory.

Parameters: Fname .string. contains the macro file name.  The
            extension ".cm" is assumed if no extension is given.
            BRIEF searches the path given by the BPATH environment
            variable if no path is specified. Macro1, macro2 ...
            .strings. are the names of the macros/globals.

   Returns: Nothing
See Also: load_macro
