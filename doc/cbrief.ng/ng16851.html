window_color ([color])

     Usage: Set background color of borderless windows.

Parameters: Color .int. is the color to use.  If omitted, the next
            color (after the current color) is used.  See color.
            Changing the background to the same as the foreground is
            an error.

   Returns: Non-0 for success, 0 for failure.
See Also: borders inq_window_color color
