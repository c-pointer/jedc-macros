close_window([win_id])

     Usage: Closes a window.  The adjoining windows are enlarged to
            take up the space.  The first choice of borders to
            eliminate is a border with a window sharing an
            identically sized edge.  If there is more than one, the
            left is eliminated first, then right, bottom and top.
            The top left window becomes current.

Parameters: Win_id .int. is the window id to close.  If omitted, the
            current window is closed.

   Returns: Nothing
See Also: create_edge delete_window inq_window move_edge
