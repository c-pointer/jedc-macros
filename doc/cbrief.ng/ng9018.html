getwd (disk_letter, result)

     Usage: Gets the working directory.

Parameters: Disk_letter .string. contains drive letter (NULL = current
            drive). Result .string. is filled in with the requested
            directory path (including drive letter).

   Returns: Greater than 0 for success, 0 for failure.
See Also: dos del date exist getwd time
