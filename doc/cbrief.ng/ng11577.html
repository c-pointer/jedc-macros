returns (val)

     Usage: Sets the return value to be passed back to the calling
            function.  This DOES NOT RETURN IMMEDIATELY, it only sets
            the return value.  If neither returns or return are
            used, the return value is undefined.

Parameters: Val .int or string. is the value to be returned to the
            caller.

   Returns: Nothing
