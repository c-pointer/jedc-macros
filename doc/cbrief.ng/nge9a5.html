next_buffer ([system_too])

     Usage: Gets the next buffer id from the (circular) buffer list.
            This only gets the next buffer number, it does change the
            current buffer.

Parameters: System_too .int. is non-0 to include system buffers,
            and 0 or omitted to skip over them.

   Returns: The next buffer id .int. in the buffer list.
See Also: attach_buffer delete_buffer inq_buffer set_buffer
