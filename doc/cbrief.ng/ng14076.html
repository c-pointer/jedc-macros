set_mouse_type(typ, [button1_loc])

     Usage: Sets the mouse type.

Parameters: Type .int. gives the mouse type:
                        0 - None
                        2 - 2 buttons
                        3 - 3 buttons
            Button1_loc .int. tells which side of the mouse should be
            called "button 1".  0 means "left-most", 1 means
            "right-most".  The default is "left-most".

   Returns: Nothing
See Also: set_btn2_action inq_btn2_action set_mouse_action inq_mouse_action
