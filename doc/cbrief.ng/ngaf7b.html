inq_kbd_char ()

     Usage: Get the status of the keyboard input buffer.  Keys get
            there from either the user pressing them or the
            push_back macro.

Parameters: None

   Returns: 0 if buffer empty, non-0 otherwise.
See Also: inq_kbd_flags keyboard_flush read_char
