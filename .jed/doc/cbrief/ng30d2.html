attach_buffer (buf_id)

     Usage: Attaches a buffer to the current window.  DANGER: if the
            current buffer is not attached to the current window when
            a screen refresh occurs, who knows what will happen!
            +--------------------------------------------------------+
            |WARNING: Hazardous to your program if incorrectly used. |
            +--------------------------------------------------------+

Parameters: Buf_id .int. is a buffer id.  Valid buffer id's come only
            from inq_buffer or create_buffer.


   Returns: Nothing
See Also: create_buffer inq_buffer create_window set_buffer
