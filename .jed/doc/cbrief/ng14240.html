set_msg_level (message_level)

     Usage: BRIEF can suppress certain types of messages.  The
            message levels are:
              0  all messages on (normal)
              1  normal messages suppressed (from primitives)
              2  error messages suppressed
              3  error and normal messages suppressed
            During commands from the keyboard or during registered
            macros, BRIEF automatically sets level 1.  The level returns
            to level 0 on return.  Only message level 3 will suppress
            messages from message and error functions.

Parameters: Message_level .int. is the new message level.

   Returns: The previous message level.
See Also: error inq_msg_level message
