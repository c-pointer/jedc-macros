inq_kbd_flags ()

     Usage: Get the state of the special keyboard flags.

Parameters: None

   Returns: The keyboard flags .int..  Each key is assigned to a
            bit:
                         0x1        Right Shift
                         0x2        Left Shift
                         0x4        Ctrl
                         0x8        Alt
                         0x10       Scroll
                         0x20       Num Lock
                         0x40       Caps Lock
See Also: inq_kbd_char
