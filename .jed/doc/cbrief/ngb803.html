inq_marked ([start_line], [star_col], [end_line], [end_col])

     Usage: Get the boundaries and type of a marked area.

Parameters: All are .ints..  Start_line, start_col, end_line and
            end_col are set to the boundaries of the marked area.

   Returns: The mark type or 0 if no marked area exists:
                        0    No mark
                        1    normal mark
                        2    column mark
                        3    line mark
                        4    noninclusive mark
See Also: drop_anchor inq_mark_size mark raise_anchor
