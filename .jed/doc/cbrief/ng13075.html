set_buffer (buf_id)

     Usage: Makes a new buffer, specified by buf_id, current.  This
            does not affect the windows on screen or any attachment to
            a particular window. Registered macros are not called when
            the buffer is changed with set_buffer and attach_buffer.
            DANGER: be sure the current buffer is attached to the
            current window when process is called or the macro exits.
            +--------------------------------------------------------+
            |WARNING: Hazardous to your program if incorrectly used. |
            +--------------------------------------------------------+

Parameters: Buf_id .int. is the new buffer id.

   Returns: Previous buffer id.
See Also: attach_buffer create_buffer inq_buffer
