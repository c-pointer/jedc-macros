undo ()

     Usage: Undoes the last command from the keyboard. From a macro,
            undo will only work of the buffer has not been modified
            since the last command.  Undo information is lost when a
            buffer is written to disk.  Undo information is kept for
            each buffer. If a command does not either change the current
            position, change a mark, or change the text, then it cannot
            be undone.

Parameters: None

   Returns: Nothing
See Also: redo
