tabs (tab1, tab2, {tab3} ...)

     Usage: Set tab stops.  The settings apply only to the current
            buffer.  Column numbers start at 1, so tab stops can be
            anywhere from 2 to the max line length.  Tabs beyond the
            last specified value are set according to the distance
            between the previous two.

Parameters: The tab stop values.  They must be in ascending order.

   Returns: Nothing
See Also: distance_to_tab use_tab_char
