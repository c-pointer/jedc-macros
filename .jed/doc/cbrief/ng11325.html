restore_position ([move])

     Usage: Restores a position saved by save_position. Positions are
            saved on a stack, so this is a pop operation. The
            position is restored in the buffer it was saved, but if
            this is not the current buffer, the buffer will NOT
            change.

Parameters: Move .int. is non-0 or omitted if the cursor should
            actually move to the old position, 0 if the stacked
            position should just be discarded.

   Returns: Greater than 0 for success, 0 or less for failure
See Also: inq_buffer move_abs move_rel save_position
