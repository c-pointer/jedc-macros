mark ([type])

     Usage: Toggle the anchor status. If another type of anchor is
            already dropped, its type is changed to the new type.

Parameters: Type .int. is the mark type. If omitted, a normal mark is
            used.
                        1    normal mark
                        2    column mark
                        3    line mark
                        4    noninclusive mark

   Returns: Nothing
