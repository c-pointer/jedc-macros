prev_char ([number])

     Usage: Moves the cursor backward one or more characters.  Unlike
            backspace, this function crosses line boundaries.  Tabs
            are treated as single characters.

Parameters: Number .int. is the number of characters to move.

   Returns: 0 if failure (no movement), non-0 if success
See Also: move_abs move_rel next_char
