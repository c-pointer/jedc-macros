inq_scrap ([last_newline], [mark_type])

     Usage: Get scrap buffer id and mark type.  This id is only valid
            until the next cut or copy...then it becomes invalid.

Parameters: Last_newline .int. is set to non-0 if the last newline
            in the scrap should be included, 0 if it should be
            ignored (see set_scrap_info). Mark_type .int. is set to
            the mark type for the scrap
                        1    normal mark
                        2    column mark
                        3    line mark
                        4    noninclusive mark

   Returns: The buffer id .int. of scrap.
See Also: copy cut paste set_scrap_info
