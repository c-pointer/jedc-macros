inq_position ([line], [col])

     Usage: Get current position in the buffer. The first character
            of the buffer is 1,1.

Parameters: Line and col .ints. are filled with the current line and
            column.

   Returns: 0 if not past end of buffer.  The the number of lines
            past end of buffer otherwise.
