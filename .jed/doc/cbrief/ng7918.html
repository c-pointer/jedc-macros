edit_file ({filename})

     Usage: Bring a file into the current buffer (and window) for
            editing.  All type 1 and type 6 registered macros and file
            extension macros are called. If filename is already in another
            buffer, that buffer becomes current.

Parameters: Filename .string. contains the name of the file to edit.

   Returns: 1 if the buffer already existed, 2 if the buffer was
            newly created, 0 or less for failure.
See Also: attach_buffer create_buffer set_buffer
